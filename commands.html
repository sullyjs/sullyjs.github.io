General Commands

Below you can find a table of XtraCord bot commands.


Commands Description Usage
Backup Manage your server backups using the following commands. x?backup
Auto Backup Manage auto backups using these commands. x?backup
AntiNuke Setup the Anti-Nuke system on your server. x?antinuke
Clone Clone a discord server. x?clone
Templates Group of commandsto submit and load pre-made templates. x?template
Premium Sends a link to buy XtraCord premium. x?premium
About Shows information about the bot. x?about
Help Sends a help message x?help
Stats Shows general bot stats x?stats
Support Sends an invite link to invite the bot. x?support
Invite Sends an invite link to invite the bot. x?invite
Suggest Suggest a feature which could be implemented on the bot. x?userinfo
Updates See the updates to the bot x?updates
Reportbug Report a bug to the developers. x?reportbug
Copy Copy emojis, channels or roles from a different server. x?copy
